# Si Bujur (Sistem Bukti & Jejak Surat)

**By Mas Alfy**

## Tentang Aplikasi

**Si Bujur** adalah aplikasi manajemen persuratan modern yang dirancang tidak hanya untuk mencatat surat masuk dan keluar, tetapi juga memiliki fitur unggulan untuk **verifikasi dan validasi surat keluar**.

### Fitur Utama

*   **Manajemen Surat Masuk & Keluar**: Pencatatan dan pelacakan surat yang efisien.
*   **Verifikasi & Validasi**: Alur kerja persetujuan berjenjang untuk surat keluar sebelum diterbitkan.
*   **Pengarsipan Cloud**: Seluruh arsip surat tersimpan aman dan terintegrasi langsung dengan **Google Drive**.

## Teknologi

Project ini dibangun menggunakan:
*   [Next.js](https://nextjs.org/)
*   TypeScript
*   Google Drive API / Apps Script (untuk penyimpanan)
